# Задача: Нормализация пробелов

> **Уровень сложности:** Easy  
> **Тема:** Строки, один проход  
> **Встречается на:** Яндекс, Microsoft

---

## 1. Что просят?

```
Дана строка. Заменить все последовательности пробелов на один пробел.
Убрать пробелы в начале и конце.

Примеры:
  "  hello   world  " → "hello world"
  "a  b  c"           → "a b c"
  "   "               → ""
```

---

## 2. Ключевая идея

Один проход. Флаг `prevWasSpace` показывает, был ли предыдущий записанный символ пробелом.
Если встречаем пробел и `prevWasSpace=true` — пропускаем (это дубликат).
Начинаем с `prevWasSpace=true` — чтобы срезать ведущие пробелы.

---

## 3. Пошаговый алгоритм

```
prevWasSpace = true  (чтобы убрать пробелы в начале)
writePos = 0

Для каждого символа c:
  если c == ' ':
    если prevWasSpace == false:
      запись[writePos++] = ' '
      prevWasSpace = true
    // иначе — пропускаем дубликат
  иначе:
    запись[writePos++] = c
    prevWasSpace = false

Если последний записанный символ — пробел: writePos--
Вернуть chars[0..writePos]
```

---

## 4. Трассировка

**Вход:** `"  hello   world  "`

```
prevWasSpace=true
i=0: ' ' пробел, prevWasSpace=true → пропускаем
i=1: ' ' пробел, prevWasSpace=true → пропускаем
i=2: 'h' → пишем 'h', writePos=1, prev=false
i=3: 'e' → пишем 'e', writePos=2
...
i=7: ' ' пробел, prev=false → пишем ' ', writePos=6, prev=true
i=8: ' ' пробел, prev=true → пропускаем
i=9: ' ' пробел, prev=true → пропускаем
i=10: 'w' → пишем 'w', writePos=7, prev=false
...
i=15: ' ' пробел, prev=false → пишем ' ', writePos=12, prev=true
i=16: ' ' пробел, prev=true → пропускаем

После цикла: writePos=12, prevWasSpace=true → writePos=11

Результат: "hello world" ✓
```

---

## 5. Сложность

| | Сложность |
|---|---|
| Время | O(n) |
| Память | O(1) — если работаем in-place по массиву символов |

---

## 6. Типичные ошибки

**Ошибка 1: Не убрать ведущие пробелы.**
Инициализировать `prevWasSpace = true` — это автоматически пропустит пробелы в начале.

**Ошибка 2: Не убрать хвостовой пробел.**
После цикла проверить: если `prevWasSpace=true` и `writePos > 0`, уменьшить `writePos` на 1.

**Ошибка 3: Использовать `split` + `join` — это O(n) памяти.**
```dart
s.trim().split(RegExp(r'\s+')).join(' ') // работает, но не in-place
```
