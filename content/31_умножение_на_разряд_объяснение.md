# Задача: Умножение большого числа на однозначный множитель

> **Уровень сложности:** Easy-Medium  
> **Тема:** Строки, арифметика с переносом  
> **Встречается на:** Яндекс

---

## 1. Что просят?

```
Большое число задано строкой цифр в формате "младший разряд первый".
Умножить его на однозначный множитель (1-9).
Вернуть результат в том же формате.

Пример:
  "321" = число 123 (3 — единицы, 2 — десятки, 1 — сотни)
  "321" * 2 = 246 → "642"
```

---

## 2. Ключевая идея

Один проход слева направо (по цифрам от мл. к ст.). На каждом шаге умножаем цифру на множитель, добавляем перенос. Новый перенос = произведение / 10. Результат: произведение % 10.

---

## 3. Пошаговый алгоритм

```
carry = 0

Для каждой цифры d в number (слева направо):
  product = d * digit + carry
  result.append(product % 10)
  carry = product / 10

Пока carry > 0:
  result.append(carry % 10)
  carry /= 10
```

---

## 4. Трассировка

**"321" * 3 (число 123 * 3 = 369 → "963")**

```
i=0: d=3, product=3*3+0=9, result="9", carry=0
i=1: d=2, product=2*3+0=6, result="96", carry=0
i=2: d=1, product=1*3+0=3, result="963", carry=0

Ответ: "963" ✓ (369 в формате мл-первый)
```

**"99" * 9**

```
i=0: d=9, product=81, result="1", carry=8
i=1: d=9, product=81+8=89, result="19", carry=8
carry>0: result="198", carry=0

Число "99" = 99. 99*9=891. В мл-первом: "198" ✓
```

## 6. Типичные ошибки

**Ошибка 1: Перевернуть строку перед обработкой.**
Строка уже в правильном порядке (мл. разряд первый) — переворачивать не нужно.

**Ошибка 2: Не дописать carry после основного цикла.**
Если после последней цифры carry > 0 — это старшие разряды результата.
