# Задача: Разворачивание скобочной грамматики

> **Уровень сложности:** Medium-Hard  
> **Тема:** Рекурсия, стек, парсинг  
> **Встречается на:** Яндекс (любимая задача!), Google

---

## 1. Что просят?

```
Дана строка по правилам грамматики:
  term ::= a-z           (одна буква)
  term ::= term term     (конкатенация)
  term ::= [term N]      (повторить N раз)

Развернуть строку в обычную.

Примеры:
  "ab"           →  "ab"
  "[ab2]"        →  "abab"
  "[a[b2]3]"     →  "abbabbabb"
  "[[ab2][c3]2]" →  "ababcccababccc"
```

## 2. Как думать на собесе

### Шаг 1: Распознай тип задачи
> "Это задача на рекурсивный парсинг. Структура вложенная — значит рекурсия или стек."

### Шаг 2: Опиши рекурсивную структуру вслух
> "Функция разбирает строку начиная с позиции i и возвращает результат + новую позицию.
> Если вижу '[' — рекурсивно разбираю содержимое до ']', потом читаю число, повторяю.
> Если вижу букву — просто добавляю её."

### Шаг 3: Определи сигнатуру функции
```dart
// Вспомогательная функция возвращает (результат, новая позиция)
(String, int) _parse(String s, int i)
```

## 3. Код на Dart

```dart
String expandGrammar(String s) {
  final (result, _) = _parse(s, 0);
  return result;
}

(String, int) _parse(String s, int i) {
  final result = StringBuffer();

  while (i < s.length && s[i] != ']') {
    if (s[i] == '[') {
      // Входим в скобки — рекурсивно разбираем содержимое
      final (inner, newI) = _parse(s, i + 1);
      i = newI + 1; // пропускаем ']'

      // Читаем число после ']'
      int count = 0;
      while (i < s.length && _isDigit(s[i])) {
        count = count * 10 + int.parse(s[i]);
        i++;
      }
      if (count == 0) count = 1; // число не указано

      // Повторяем inner count раз
      for (int k = 0; k < count; k++) {
        result.write(inner);
      }
    } else {
      // Обычная буква
      result.write(s[i]);
      i++;
    }
  }

  return (result.toString(), i);
}

bool _isDigit(String c) => c.codeUnitAt(0) >= 48 && c.codeUnitAt(0) <= 57;
```

## 4. Трассировка на примере

**Вход:** `"[a[b2]3]"`

```
_parse(s, 0):
  i=0: '[' → рекурсия _parse(s, 1)
    _parse(s, 1):
      i=1: 'a' → result="a", i=2
      i=2: '[' → рекурсия _parse(s, 3)
        _parse(s, 3):
          i=3: 'b' → result="b", i=4
          i=4: ']' → выходим
        возврат ("b", 4)
      i=5 (после ']'), читаем число: '2' → count=2, i=6
      повторяем "b" 2 раза → result="abb"
      i=6: ']' → выходим
    возврат ("abb", 6)
  i=7 (после ']'), читаем число: '3' → count=3, i=8
  повторяем "abb" 3 раза → "abbabbabb"

Ответ: "abbabbabb" ✓
```

## 5. Частые ошибки

**Ошибка 1: Не двигать индекс после ']'.**
```dart
final (inner, newI) = _parse(s, i + 1);
i = newI + 1; // ← "+1" чтобы пропустить сам ']'
```

**Ошибка 2: Не обработать многозначные числа.**
```dart
// Плохо — работает только для 0-9
count = int.parse(s[i]);

// Хорошо — работает для 10, 100, ...
while (i < s.length && _isDigit(s[i])) {
  count = count * 10 + int.parse(s[i]);
  i++;
}
```

**Ошибка 3: Паниковать когда интервьюер спрашивает про стек.**  
Скажи: "Рекурсия — это неявный стек. Если нужно без рекурсии — переписываю явно через Stack."

## 6. Версия без рекурсии — через стек

```dart
String expandGrammarStack(String s) {
  final stack = <(StringBuffer, int)>[]; // (накопленная строка, кол-во повторений)
  var current = StringBuffer();

  int i = 0;
  while (i < s.length) {
    if (s[i] == '[') {
      stack.add((current, 0));
      current = StringBuffer();
      i++;
    } else if (s[i] == ']') {
      i++;
      int count = 0;
      while (i < s.length && _isDigit(s[i])) {
        count = count * 10 + int.parse(s[i]);
        i++;
      }
      if (count == 0) count = 1;
      final inner = current.toString();
      final (prev, _) = stack.removeLast();
      current = prev;
      for (int k = 0; k < count; k++) {
        current.write(inner);
      }
    } else {
      current.write(s[i]);
      i++;
    }
  }

  return current.toString();
}
```
