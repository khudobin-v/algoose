# Задача: Количество подстрок без повторяющихся символов

> **Уровень сложности:** Medium  
> **Тема:** Скользящее окно (Sliding Window), HashMap  
> **Аналог на LeetCode:** 3. Longest Substring Without Repeating Characters  
> **Встречается на:** Яндекс, все крупные компании

---

## 1. Что просят?

```
Дана строка символов.
Найти количество пар индексов [i, j] (i <= j), между которыми
включительно нет повторяющихся символов.

Примеры:
  "aab"   →  5   // пары: [0,0]="a", [1,1]="a", [2,2]="b", [1,2]="ab", [0,0]="a"
             // валидные без повторов: "a"(0), "a"(1), "b"(2), "ab"(1-2) = 4? 
             // Нет: [0,0], [0,1] - "aa" невалид, [1,1], [1,2]="ab", [2,2] = 4
  "abcd"  →  10  // все подстроки валидны: C(4,2)+4 = 6+4 = 10
```

> ⚠️ Читай условие внимательно: просят **количество пар**, а не длину самой длинной подстроки.

---

## 2. Как думать на собесе

### Шаг 1: Пойми что считаем
Для каждого правого конца `j` — сколько левых концов `i` дают валидную строку?  
Это все `i` от `left` до `j`, где `left` — начало текущего окна без повторов.

Количество валидных пар с правым концом `j` = `j - left + 1`.

### Шаг 2: Алгоритм скользящего окна
```
left = 0
seen = Map<символ, последняя позиция>
result = 0

Для каждого j:
  если s[j] уже видели и его позиция >= left:
    двигаем left = позиция_повтора + 1
  
  обновляем seen[s[j]] = j
  result += j - left + 1
```

---

## 3. Код на Dart

```dart
int countSubstringsNoRepeat(String s) {
  final lastSeen = <String, int>{};
  int left = 0;
  int result = 0;

  for (int j = 0; j < s.length; j++) {
    final char = s[j];

    // Если символ был в текущем окне — сдвигаем левую границу
    if (lastSeen.containsKey(char) && lastSeen[char]! >= left) {
      left = lastSeen[char]! + 1;
    }

    lastSeen[char] = j;

    // Количество валидных подстрок с правым концом j
    result += j - left + 1;
  }

  return result;
}
```

## 4. Трассировка

**Вход:** `"aab"`

```
left=0, seen={}, result=0

j=0: char='a', not in seen → seen={'a':0}, result += 0-0+1 = 1  → result=1
j=1: char='a', seen['a']=0 >= left=0 → left=1, seen={'a':1}, result += 1-1+1=1 → result=2
j=2: char='b', not in seen → seen={'a':1,'b':2}, result += 2-1+1=2 → result=4

Ответ: 4 ✓
```

**Проверим вручную:**
```
[0,0]="a" ✓
[0,1]="aa" ✗ (повтор)
[0,2]="aab" ✗ (повтор)
[1,1]="a" ✓
[1,2]="ab" ✓
[2,2]="b" ✓

Итого: 4 валидных пары ✓
```

## 5. Частые ошибки

**Ошибка 1: Двигать left когда символ был РАНЬШЕ левой границы.**  
```dart
// Плохо — left прыгнет назад!
if (lastSeen.containsKey(char)) left = lastSeen[char]! + 1;

// Хорошо — только если повтор в текущем окне
if (lastSeen.containsKey(char) && lastSeen[char]! >= left) {
  left = lastSeen[char]! + 1;
}
```

**Ошибка 2: Путать "длину самой длинной подстроки" и "количество подстрок".**  
LeetCode 3 просит длину. Эта задача просит количество. `result += j - left + 1` — для количества.

## 6. Вариант: найти ДЛИНУ самой длинной подстроки

```dart
int lengthOfLongestSubstring(String s) {
  final lastSeen = <String, int>{};
  int left = 0;
  int maxLen = 0;

  for (int j = 0; j < s.length; j++) {
    final char = s[j];
    if (lastSeen.containsKey(char) && lastSeen[char]! >= left) {
      left = lastSeen[char]! + 1;
    }
    lastSeen[char] = j;
    maxLen = maxLen > j - left + 1 ? maxLen : j - left + 1;
  }

  return maxLen;
}
```
