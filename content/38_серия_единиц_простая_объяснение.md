# Задача: Максимальная серия единиц

> **Уровень сложности:** Easy
> **Тема:** Массивы, один проход
> **Аналог на LeetCode:** 485. Max Consecutive Ones

## 1. Что просят?

```
Дан бинарный массив nums (состоит только из 0 и 1).
Найти длину максимальной непрерывной последовательности единиц.

Пример:
  nums = [1,1,0,1,1,1]  →  3
  nums = [1,0,1,1,0,1]  →  2
```

## 2. Идея

Один проход: ведём счётчик `cur` текущей серии единиц и `max` — лучший результат.

- Если `nums[i] == 1` → `cur++`, обновляем `max`
- Если `nums[i] == 0` → сбрасываем `cur = 0`

## 3. Алгоритм

```
cur = 0, maxLen = 0
для каждого x в nums:
    если x == 1:
        cur++
        maxLen = max(maxLen, cur)
    иначе:
        cur = 0
вернуть maxLen
```

## 4. Сложность

- **Время:** O(n) — один проход
- **Память:** O(1) — только два счётчика

## 5. Финальный код Dart

```dart
int findMaxConsecutiveOnes(List<int> nums) {
  int cur = 0, maxLen = 0;
  for (final x in nums) {
    if (x == 1) {
      cur++;
      if (cur > maxLen) maxLen = cur;
    } else {
      cur = 0;
    }
  }
  return maxLen;
}
```
