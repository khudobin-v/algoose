# Задача: Минимальное расстояние между символами X и Y

> **Уровень сложности:** Easy  
> **Тема:** Один проход, два указателя  
> **Встречается на:** Яндекс

---

## 1. Что просят?

```
Дана строка из символов 'X', 'Y', 'O'.
Найти минимальное расстояние (по индексу) между любым X и любым Y.

Примеры:
  "XOOOY"   → 4  (X на 0, Y на 4)
  "XOOYOYX" → 1  (Y на 5, X на 6)
  "XY"      → 1
```

---

## 2. Ключевая идея

Один проход. Запоминаем позицию последнего встреченного X и последнего Y.
При каждом новом X или Y считаем расстояние до последнего символа другого типа.
Минимальная пара всегда состоит из двух соседних (по позиции) X и Y.

---

## 3. Пошаговый алгоритм

```
lastX = null, lastY = null, minDist = infinity

Для каждого i, символ c:
  если c == 'X':
    lastX = i
    если lastY != null: minDist = min(minDist, |lastX - lastY|)
  если c == 'Y':
    lastY = i
    если lastX != null: minDist = min(minDist, |lastY - lastX|)

Вернуть minDist
```

---

## 4. Трассировка

**Вход:** `"XOOYOYX"`

```
i=0: 'X' → lastX=0, lastY=null → нет обновления
i=1: 'O' → пропуск
i=2: 'O' → пропуск
i=3: 'Y' → lastY=3, lastX=0 → dist=3, minDist=3
i=4: 'O' → пропуск
i=5: 'Y' → lastY=5, lastX=0 → dist=5, minDist=3 (не лучше)
i=6: 'X' → lastX=6, lastY=5 → dist=1, minDist=1

Ответ: 1 ✓
```

---

## 5. Сложность

| | Сложность |
|---|---|
| Время | O(n) |
| Память | O(1) |

---

## 6. Типичные ошибки

**Ошибка 1: Искать ближайший Y для каждого X отдельным циклом — O(n²).**
Один проход с запоминанием последней позиции — O(n).

**Ошибка 2: Не обработать случай когда X или Y отсутствуют в строке.**
Проверить наличие обоих символов перед началом, или вернуть null/специальное значение.

**Ошибка 3: Считать расстояние только в одну сторону.**
Нужно сравнивать каждый новый X с последним Y И каждый новый Y с последним X.
