# Задача: Найти все анаграммы строки s в строке t

> **Уровень сложности:** Medium  
> **Тема:** Скользящее окно, частотный массив  
> **Аналог на LeetCode:** 438. Find All Anagrams in a String  
> **Встречается на:** Яндекс, Google, Facebook

---

## 1. Что просят?

```
Даны строки t и s. Найти все стартовые индексы подстрок t,
которые являются анаграммами строки s.
Анаграмма — строка, содержащая те же символы в любом порядке.

Примеры:
  t="cbaebabacd", s="abc" → [0, 6]
    "cba" (0-2) — анаграмма "abc" ✓
    "bae" (1-3) — нет
    "eba" (4-6) — нет
    "bac" (5-7) — нет
    "abc" (6-8) — анаграмма "abc" ✓
  t="abab", s="ab" → [0, 1, 2]
```

---

## 2. Ключевая идея

Скользящее окно длины `s.length` + частотные массивы букв.
Вместо того чтобы сортировать каждую подстроку (O(k log k)), храним частоты символов.
При сдвиге окна: добавляем новый правый символ, удаляем старый левый — O(1) на шаг.

---

## 3. Пошаговый алгоритм

```
1. Создать частотный массив sCount для строки s
2. Создать частотный массив windowCount для первого окна t[0..len(s)-1]
3. Если sCount == windowCount → индекс 0 в ответе
4. Сдвигать окно вправо (i от len(s) до len(t)-1):
   - добавить t[i] в windowCount
   - убрать t[i - len(s)] из windowCount
   - если sCount == windowCount → добавить (i - len(s) + 1) в ответ
```

---

## 4. Сложность

| | Сложность |
|---|---|
| Время | O(n), где n = len(t) |
| Память | O(1) — массивы размером 26 |

---

## 5. Типичные ошибки

**Ошибка 1: Сортировать подстроку на каждом шаге — O(n*k log k).**
Используй частотный массив — O(n).

**Ошибка 2: Неправильно вычислять левую границу окна.**
```dart
// При правой границе i, левая граница = i - s.length
// Новое начало окна = i - s.length + 1
result.add(i - s.length + 1); // не i - s.length!
```

**Ошибка 3: Создавать новый Map на каждой итерации.**
Обновляй один и тот же массив — это O(1), а не O(k).

---

## 6. Псевдокод

```
sCount[26] = частоты(s)
windowCount[26] = частоты(t[0..len(s)-1])

если sCount == windowCount: result.add(0)

для i от len(s) до len(t)-1:
  windowCount[t[i]]++
  windowCount[t[i - len(s)]]--
  если sCount == windowCount: result.add(i - len(s) + 1)
```
