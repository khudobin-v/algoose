# Задача: Удаление смайликов из строки

> **Уровень сложности:** Easy-Medium  
> **Тема:** Строки, конечный автомат, регулярные выражения  
> **Встречается на:** Яндекс

---

## 1. Что просят?

```
Удалить из строки все вхождения ":-)" и ":-(".

Примеры:
  "hello :-)world"  → "hello world"
  ":-) :-(hi"       → " hi"
  "a:-):-(b"        → "ab"
```

---

## 2. Два подхода

**Простой:** регулярное выражение `:-[)(]` — одна строка кода.

**Без regex — конечный автомат:** три состояния.

---

## 3. Конечный автомат

```
Состояние 0 (обычный режим):
  ':' → переходим в состояние 1 (запоминаем ':')
  иначе → выводим символ

Состояние 1 (видели ':'):
  '-' → переходим в состояние 2
  ':' → выводим предыдущий ':', остаёмся в состоянии 1
  иначе → выводим ':', выводим текущий, → состояние 0

Состояние 2 (видели ':-'):
  ')' или '(' → СМАЙЛИК НАЙДЕН, пропускаем, → состояние 0
  ':' → выводим ':-', → состояние 1
  иначе → выводим ':-', выводим текущий, → состояние 0
```

---

## 4. Сложность

| Подход | Время | Память |
|---|---|---|
| Regex | O(n) | O(n) |
| Автомат | O(n) | O(1) |

---

## 5. Типичные ошибки

**Ошибка 1: Не обработать незавершённые смайлики в конце строки.**
Если строка заканчивается на `":"` или `":-"` — нужно дописать их в результат.

**Ошибка 2: Обрабатывать `":)"` или `":-)"` как одно и то же.**
Задача просит удалять только `":-)"` с тире, не `":)"`.

---

## 6. Псевдокод автомата

```
state = 0
для каждого c:
  если state==0 и c==':': state=1
  если state==1 и c=='-': state=2
  если state==2 и c in ')(',: state=0 (смайлик удалён)
  иначе: вывести накопленные символы + c, state=0
в конце: вывести накопленные символы
```
